API Information

This is a demo of the YouTube to MP3 Converter API. The API provides:

  • POST /api/convert - Convert YouTube video to MP3
  • GET /api/status/{task_id} - Check conversion status
  • GET /api/download/{filename} - Download converted file
  • GET /api/health - Health check

For production use, deploy this API to Railway and use the provided URL.
